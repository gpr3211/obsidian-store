### Commands aws

```
aws configure

```
send contents of bucket to tmp file
```
aws s3 ls BUCKET_NAME > /tmp/bucket_contents.txt
```

### Policy
## add this policy to allow access to url retrieval
```json
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::BUCKET_NAME/*"
    }
  ]
}
```
