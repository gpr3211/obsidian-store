### File bundling

1. We can embed a file using the fyne tool
```Go
//go:generate fyne bundle -o bundled.go assets
```
This line on top of theme.go file will bundle the assets folder and generate code 
To do this 
```
go generate
```
in console and in root project dir.
```Go

func makeBanner() fyne.CanvasObject {

	toolbar := widget.NewToolbar(
		widget.NewToolbarAction(theme.HomeIcon(), func() {}),
	)
	logo := canvas.NewImageFromResource(resourceMainLogoPng)
	// by def all widgets fill all available space. We change the scale mode
	logo.FillMode = canvas.ImageFillContain
	return container.NewStack(toolbar, logo)

}
```
